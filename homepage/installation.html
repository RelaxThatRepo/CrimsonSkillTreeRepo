Installation Guide

This guide will walk you through the process of installing the Crimson Skill Tree plugin into your Unreal Engine project.

Video Tutorial

Step-by-Step Instructions

1. Download the plugin from the marketplace.

2. Locate your Unreal Engine project directory.

3. Create a "Plugins" folder if one does not already exist.

4. Unzip and copy the CrimsonSkillTree folder into the "Plugins" directory.

5. Restart the Unreal Editor. The plugin should now be active.
