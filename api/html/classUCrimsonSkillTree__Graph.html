Detailed Description

Represents the editor graph for a Crimson Skill Tree asset (UCrimsonSkillTree). This UEdGraph derivative manages the collection of graph nodes (UCrimsonSkillTree_GraphNode) and graph edges (UCrimsonSkillTree_GraphEdge) that visually define the skill tree structure in the editor. It is responsible for rebuilding the runtime UCrimsonSkillTree from this editor representation.
