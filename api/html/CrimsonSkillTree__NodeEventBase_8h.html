class  UCrimsonSkillTree_NodeEventBase
 Base class for events that are triggered by changes in a skill node's state (e.g., level up, level down, reset). These events can implement various gameplay logic such as applying gameplay effects, granting abilities, modifying attributes, or triggering other game systems. This class is intended to be subclassed in C++ or Blueprint to create specific node events. Instances are typically created as subobjects of UCrimsonSkillTree_Node. More...
